# SkillCraft Stopwatch ⏱️

A modern, interactive, and visually stunning **stopwatch web application** built with HTML, CSS, and JavaScript.

## Features

- Start, Pause, Resume, Reset
- Lap timing (with lap duration + total time)
- Millisecond precision
- Sound effects on actions
- Dark / Light theme toggle (persists in browser)
- Export laps as CSV file
- Keyboard shortcuts (Space = Start/Pause, L = Lap, R = Reset, T = Theme, E = Export)
- Auto-saves laps using localStorage
- Futuristic cyberpunk design with animations & glow effects
- Responsive & mobile-friendly

## Technologies

- HTML5
- CSS3 (Glassmorphism, gradients, animations)
- Vanilla JavaScript (no frameworks)
- Web Audio API (for beeps)
- Google Fonts (Orbitron)
